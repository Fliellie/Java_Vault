# while
```
public class HelloWorld{

     public static void main(String []args){
        int i =0;
        while(i<10){
            System.out.println(i);
            i+=1;
        }
     }
}

```

# Do while
```
public class HelloWorld{

     public static void main(String []args){
        int i = 10;
        do {
            System.out.println(i);
            i+=1;
        } while(i<10);
     }
}

```
