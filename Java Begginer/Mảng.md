mảng là kiểu dữ liệu tham chiếu tức là địa chỉ của mảng được lưu trong stack còn toàn bộ nội dung mảng được lưu trong heap

## Khai báo mảng 1 chiều
```
int [] a; // khai báo 
a= new int [3]; // tạo mảng int kích thước 3
<kiểu dữ liệu>[]<tên mảng>=new<kiểu dữ liệu>[kích cỡ mảng];
```
## in kích cỡ mảng
```
a.length
```
## Khởi tạo giá trị cho mảng
```
int [] a ={1,2,3};

```
