👉 Khi bạn tạo:

Dog d1 = new Dog();  
Dog d2 = new Dog();

- `d1` là **một instance của Dog**
- `d2` là **một instance khác**

✔ Hai instance này **độc lập với nhau**
## ⚡ So sánh nhanh

|Khái niệm|Ý nghĩa|
|---|---|
|class|khuôn mẫu|
|instance (object)|đối tượng được tạo ra|
|instance variable|biến thuộc về object|
